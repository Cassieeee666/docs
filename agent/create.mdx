## Create an Agent with AI

Solvea uses **AI to automatically create and refine your agent**. Simply describe your business needs in plain language, and Solvea will generate an agent with the appropriate role, skills, and behavior.

### Steps

<Steps>
  <Step title="Open Discovery">
    After signup, go to **Discovery** to start creating an agent.
  </Step>
  <Step title="Describe Your Needs">
    Enter your request, for example:\
    _“Create an AI receptionist for my hotel to handle bookings and service inquiries.”_
  </Step>
  <Step title="Answer Follow-up Questions">
    Solvea asks clarifying questions to complete the setup.

    - If requirements are unclear, AI will ask qustions to fill  in the gaps.
    - If a channel is unsupported, AI suggests alternatives.
    - Creation progress is shown in real time.
  </Step>
  <Step title="Preview and Refine">
    Review the agent’s personality, core skills, and sample conversations.\
    Enter additional instructions to adjust the agent if needed.
  </Step>
  <Step title="Confirm and Continue">
    Click **"Next"** to proceed to the Agent configuration page.
  </Step>
</Steps>

## Create an Agent from Templates

Solvea provides pre-built agent templates based on common business scenarios, including:

- Hotel Front Desk
- Real Estate Broker
- Clinic Receptionist
- E-commerce Support
- Software Consultant
- Hair Salon Receptionist
- Restaurant Receptionist
- Influencer Support

Each template includes the most frequently used skills for that role.

_For example, the hotel front desk agent comes with Service Booking, Service Inquiry, Check-in, and Check-out skills._

### Steps

<Steps>
  <Step title="Select a Template">
    Click **Select** to enter the Agent configuration page and start customizing.
  </Step>
</Steps>

## Agent Creation History

View a complete record of all your previously created Agents for easy tracking, traceability, and iterative modifications.

- Click **History** to access the full list of your historical Agents.
- Agents labeled **Published** are the active versions running in your production environment.
- Select any historical Agent to switch the system to that version for editing. 

<Note>
  All modifications will only take effect after you publish the Agent.
</Note>
