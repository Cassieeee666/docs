## Channels

By default, an AI agent can handle conversations via **Voice.**\
You can assign a phone number and configure the agent’s voice directly from the Agent page.

You can:

<Steps>
  <Step title="Set the agent’s phone">
    Assign a phone number to your agent so customers can reach it via voice calls.

    Learn more: [How to add a phone number](/deploy/phone/add-phone-numbers)

    <Warning>
      New users receive a free phone number valid for 7 days.\
      Please purchase or add your own phone number before it expires to avoid service interruption.
    </Warning>
  </Step>
  <Step title="Set the agent’s language and voice">
    Each phone number supports one voice.
  </Step>
  <Step title="Configure a transfer phone number">
    This is the human agent’s number that the AI calls when it cannot resolve an issue and needs live support.
  </Step>
</Steps>

<Tip>
  **Solvea agents can also handle Email and Chat.**

  Go to the Deploy Module to connect additional channels.

  Learn more: [Set up Livechat](/deploy/livechat/install-livechat) ; [Connect Email](/deploy/email)
</Tip>

## Personality

Personalize your agent’s identity and communication style.

<Steps>
  <Step title="Name and Avatar">
    Upload an avatar and set the agent’s name.

    <Info>
      We recommend using your business or brand name as the agent's name.
    </Info>
  </Step>
  <Step title="Persona">
    Define how your agent behaves.

    A clear structure works best: **role definition, tone, and general rules**. For example:

    ```
    # Role
    You are a receptionist for... You are responsible for ...
    
    # Tone
    Your style is polite, warm...
    
    # Rule
    1. Transfer to human if the user is very angry...
    2. Never...
    ```
  </Step>
  <Step title="Greeting">
    Set the first message the agent sends when answering a Call or starting a Livechat.
  </Step>
</Steps>

## Set Up Agent Skills

Complete skill configuration by connecting required channels and tools.

<Steps>
  <Step title="Select a Skill">
    Skills that require setup are clearly marked. Click to open the configuration page.
  </Step>
  <Step title="Authorize Integrations">
    Click **Connect** and follow the steps to complete channel or tool authorization.

    <Tip>
      For detailed channel and tool integration, see the [Deploy Overview](/deploy/overview).
    </Tip>
  </Step>
</Steps>

<Tip>
  To create a brand-new skill, see [Skill Overview](/skill/overview).
</Tip>
