**An AI agent is a role-based AI receptionist designed for a specific business scenario**, equipped with multiple **skills** tailored to its job.

_For example: a hotel front desk agent may include Service Booking and Service Inquiry skills._

## How an Agent Works

**AI Agent think and act like human staff.** A Solvea agent can:

- Understand user intent
- Select the right skill
- Use connected tools and channels
- Immediate handoff

This allows the agent to handle both conversations and workflows, not just reply to messages.
