## Authorization

<Steps>
  <Step>
    Enter your Shopify store domain.
  </Step>
  <Step>
    Click the **Connect** button to be redirected to Shopify.
  </Step>
  <Step>
    Login to Shopify
  </Step>
  <Step>
    In Shopify, confirm the authorization and install the app to complete setup.
  </Step>
</Steps>

## Function configuration

### Enable Livechat

Enable Livechat on Shopify for 24/7 AI agent support to resolve all customer inquiries.

<Steps>
  <Step>
    Go to Livechat → Configuration.
  </Step>
  <Step>
    Select the Livechat instance to use on your Shopify store.
  </Step>
  <Step>
    Open your Shopify Admin and navigate to **Online Store → Themes**.
  </Step>
  <Step>
    Click **Customize**, then open **App Embeds**.
  </Step>
  <Step>
    Verify that the Livechat widget appears on your store pages.
  </Step>
  <Step>
    Click **Save**.
  </Step>
</Steps>

### Sync Product Knowledge

Auto-sync Shopify product data to the knowledge base for accurate AI product recommendations and Q&A.

<Steps>
  <Step>
    Go to **Knowledge Sync → Sync**.
  </Step>
  <Step>
    After the product data is synced, click **Configure**.
  </Step>
  <Step>
    Select the folder where product knowledge will be stored.
  </Step>
  <Step>
    (Optional) Enable **Automatic Knowledge Sync**.

    <Info>
      Shopify product data is automatically updated once **per day**.
    </Info>
  </Step>
</Steps>

### Query Order Information

Retrieve Shopify order info by order number to streamline AI agent workflow execution.

<Note>
  This tool capability requires no additional configuration and is ready to use once Shopify is integrated.
</Note>

By using the Shopify  tool with an **order number**, the Agent can retrieve detailed order information, including:

- Tracking numbers
- Product details (name, SKU, quantity, price, status)
- Purchase time
- Shipping destination country

In the prompt, you can use the Shopify Order Query Tool like this:

```
Use the Order Number to call Shopify, to get the trakking numbers.
```
