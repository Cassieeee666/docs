description: "The Knowledge module allows you to upload, sync, and manage business knowledge from multiple sources. "
---

Uploaded content is automatically parsed and organized into folders, enabling agents to retrieve accurate information and respond to users more effectively.

## Upload Knowledge

Upload local documents to quickly build your knowledge base. The system will automatically extract and process the content.

### Supported File Formats

- PDF
- Word
- Excel
- CSV
- TXT

<Note>
  File size limit: 20 MB per file
</Note>

### Steps

<Steps>
  <Step title="">
    Select the folder where the knowledge should be stored. \
    Click **Add Knowledge** → **Upload Documents**.
  </Step>
  <Step title="">
    Click **Select File** and upload your document.
  </Step>
  <Step title="">
    The system checks file size automatically. Retry if any upload fails.
  </Step>
  <Step title="">
    Click **Confirm** to start the upload.
  </Step>
  <Step title="">
    The system parses the content and adds it to the selected folder.
  </Step>
</Steps>

## Sync Website Content

You can sync content directly from your website by providing a main URL. The system uses the site’s sitemap to automatically discover and categorize subpages.

### Steps

<Steps>
  <Step title="">
    Select the folder where the knowledge should be stored. \
    Click **Add Knowledge** → **Sync Website Content**.
  </Step>
  <Step title="">
    Enter the website URL.
  </Step>
  <Step title="">
    Preview detected pages and select the target directory.
  </Step>
  <Step title="">
    Click **Confirm** to start learning.
  </Step>
  <Step title="">
    The parsed website content appears in the selected folder.
  </Step>
</Steps>
