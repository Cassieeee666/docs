---

## Step 1: Start a New Skill

Click **Add New Skill** to begin creating a skill from scratch.

## Step 2: Define Basic Information

Set the **skill name , icon** and select the **service type**.

- Email
- Livechat
- Inbound Voice

<Note>
  Different service types include different configuration options.
</Note>

## Step 3: Configure Common Settings

Complete the shared configuration for all skill types:

- **When to use**

  Define the conditions under which the skill should be triggered.

  For example: "When the user expresses that he want to book an appointment."
- **Prompt**

  Write a structured skill prompt to define behavior and task logic.

  Use "`/`" to insert knowledge references, tools, or handoff actions.

  <Tip>
    Want to learn how to write a perfect prompt? See [Write a Skill Prompt](/skill/prompt)
  </Tip>
- **Knowledge Access**

  Enable knowledge-based responses and select the knowledge scope.
- **Tools**

  Add required tools and complete authorization.

## Step 4: Configure Channel-Specific Settings

Complete settings based on the skill type.

<Tabs>
  <Tab title="Email">
    - **Response Format**

      Customize the email opening and closing format.
  </Tab>
  <Tab title="Livechat">
    - **AI→human Handoff Script**

      Set up different handoff templates based on whether human agents are **in service hours.**

      <Tip>
        How to set handoff time period? See [Edit Agent Profile -\> AI Handoff Schedule](/agent/edit-agent-profile)
      </Tip>
  </Tab>
  <Tab title="Inbound Voice">
    - **AI→human Handoff Script**

      Set up different handoff templates based on whether human agents are **in service hours.**

      <Tip>
        How to set handoff time period? See [Edit Agent Profile -\> AI Handoff Schedule](/agent/edit-agent-profile)
      </Tip>
    - **Idle reminder**

      Set the **idle reminder trigger time**. If the AI’s response generation exceeds the specified threshold, it will automatically send a comfort message to calm callers.

      ### 
  </Tab>
</Tabs>

## Step 5: Save and Publish

Click **Save** to store the skill.

<Warning>
  To make the skill active in production, please “publish” the agent. Changes will only take effect after the agent is published.
</Warning>
